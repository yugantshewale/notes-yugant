### Repeating Something
To repeat something, Ruby provides us with `times`. We can use the do end block with it as shown below:
```
10.times do
  puts "I love superman"
end
```
---

### Check if the Number is Even or Odd
Ruby provides `.even?` method to check if a number is even.


Ruby also provides `.odd?` method to check if a number is odd.


```
puts 10.even?
puts 10.odd?
```

### Optional Underscore
It's hard to read a big number sometimes. We can't use a comma, since it's a number.
```
num1 = 1867496675
num2 = 186_749_6675

puts num1
puts num2

Output 
1867496675
1867496675
```
